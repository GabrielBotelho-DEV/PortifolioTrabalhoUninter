# Portfólio Pessoal - Trabalho UNINTER

## Sobre o Projeto

Este projeto foi desenvolvido como atividade prática do curso de Engenharia de Software da UNINTER.

O objetivo foi criar um site de portfólio pessoal utilizando tecnologias fundamentais do desenvolvimento web, apresentando informações profissionais, formação acadêmica, habilidades e projetos desenvolvidos.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub Pages

## Funcionalidades

* Página inicial com apresentação pessoal
* Seção sobre mim
* Exibição de habilidades técnicas
* Área de projetos desenvolvidos
* Informações de contato
* Layout responsivo para diferentes dispositivos
